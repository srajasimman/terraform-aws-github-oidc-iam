# IAM account example

Configuration in this directory sets up OIDC and IAM for GitHub Actions in your AWS accounts.

# Usage

To run this example you need to execute:

```bash
$ terraform init
$ terraform plan
$ terraform apply
```

Run `terraform destroy` when you don't need these resources.
